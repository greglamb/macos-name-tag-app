# Project Guidelines

This is a macOS SwiftUI application project.

## Installed Skills

- **swiftui-pro** — Reviews SwiftUI code for best practices on modern APIs, maintainability, and performance. Use `/swiftui-pro` to run a review.
- **swift-testing-pro** — Writes, reviews, and improves Swift Testing code using modern APIs and best practices. Use `/swift-testing-pro` to run a review.
- **superpowers** — A complete software development workflow with brainstorming, planning, execution, test-driven development, code review, and systematic debugging. Use `/brainstorm`, `/write-plan`, `/execute-plan`, and other skill commands.
