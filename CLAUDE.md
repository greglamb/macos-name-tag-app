# Project Guidelines

This is a macOS SwiftUI application project.

## Installed Skills

- **swiftui-pro** — Comprehensively reviews SwiftUI code for best practices on modern APIs, maintainability, and performance. Use `/swiftui-pro` to run a review, or `/swiftui-pro <focus area>` for a targeted check (e.g., `/swiftui-pro Check for deprecated API`).
